Common "Not Set" Issues in GA4

Many GA4 reports show "(not set)" due to missing or misconfigured parameters, affecting dimensions like source/medium, campaign, or page title.

Causes:

  • Improper tagging on website or app.
  • Missing URL parameters for campaigns.
  • Incorrect event configuration in GA4.
  • Data sampling or filters misapplied.

Solutions:

  • Verify all event tags and parameters.
  • Ensure UTM parameters are correctly applied for campaigns.
  • Use GTM or manual code to fix missing tracking.
  • Check and correct filters in GA4 reports.
